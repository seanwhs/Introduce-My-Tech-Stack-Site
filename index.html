Node.js is gone. Bun is runtime, package manager, bundler, test runner, and a native compilation target for desktop binaries.

Inngest
Nervous System — Time & Coordination

The connective tissue that transforms isolated tools into a unified distributed platform. Durable step functions via standard HTTP. No message broker overhead. No infra to maintain.

  • Durable step functions with auto-retries
  • step.sleep() & step.waitForEvent()
  • Event replay for self-healing workflows
  • Serverless — invoked via HTTP webhooks

Design Philosophy

Failure is not an
exception. It's a path.

"Most architecture diagrams assume a perfect network. This system is architected under the direct assumption that things will crash, APIs will time out, and database connections will face exhaustion."

01

Failures Became Local

A broken service doesn't collapse the system — it isolates impact. Each layer fails independently. The rest continues operating without human intervention.

02

Everything Is Replayable

Because Inngest sits at the center, all critical workflows — onboarding, billing, content sync, asset provisioning — are event-driven and fully replayable.

03

Recovery Is Automatic

Retries, exponential backoff, event replay, and compensating actions replace live debugging. The system heals itself — no pager-duty required.

The result

A portable, self-healing runtime that scales across environments without rewriting its core.
